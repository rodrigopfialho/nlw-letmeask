## Installation 

Clone the project and access the folder.

```bash 
  https://github.com/rodrigopfialho/nlw-letmeask.git
  cd letmeask
```
    
 ```bash 
  Install the dependencies:
  yarn

  # Start the project
  $ yarn start
```   
The app will be available for access on your browser at http://localhost:3000
